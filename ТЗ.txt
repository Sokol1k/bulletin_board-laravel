Доска объявлений - Laravel
Реализовать сайт "Доска объявлений для сдачи недвижимости в аренду" на Laravel.
На сайте должны быть такие страницы:
1.	Страница регистрации (нужно использовать страницу и функционал регистрации, который предоставляет Laravel)
2.	Страница авторизации (нужно использовать страницу и функционал авторизации, который предоставляет Laravel)
3.	Страница со списком объявлений (страницу видят все пользователи)
    1.	На странице должно отображаться:
        1.	Картинка объявления (если картинки нет, нужно отображать какое-то изображение по умолчанию)
        2.	Title объявления (ссылка на страницу отображения объявления)
        3.	Дата создания
        4.	Description (не боле 255 символов, если символов больше, то нужно обрезать description и добавлять троеточие)
    2.	Нужно добавить пагинацию, загружать по 15 объявлений
4.	Страница с отображением объявления (страницу видят все пользователи)
    1.	На странице должно отображаться:
        1.	Картинка объявления (если картинки нет, нужно отображать какое-то изображение по умолчанию)
        2.	Title объявления
        3.	Дата создания
        4.	Описание
        5.	Номер телефона
        6.	Страна
        7.	Email
        8.	Google карта с отображением выбранных координат
5.	Страница создания / редактирования объявления (Функционал создания доступен только для авторизированых пользователей, функционал редактирования только для автора объявления)
    1.	title *
    2.	description * (textarea)
    3.	phone *  (поле с масками под ввод телефонов, например маска для американского формата имеет следующий вид “+1 (555) 555-5555”)
    4.	country *
    5.	email *
    6.	end date *  (использовать какой то виджет/плагин для удобного выбора даты)
    7.	image (file field)
    8.	На карте должна быть возможность отметить адрес. Объявление можно сохранить не выбрав адрес
    9.	Широта и долгота выбранного адреса должна сохранятся в БД
    10.	На странице должна быть форма с такими полями:
    11.	Под формой необходимо встроить карту (Google Map)
    12.	Под картой справа нужно разместить кнопку Save, при нажатии на нее, успешной валидации данных в форме и сохранении, юзер переходит на страницу с отображением созданного объявления.

6. Сделать 2 роли админ и обычный юзер.
1. Юзео не может пользоватся сервисом пока админ его не подтверддит
2. У админа должна быть страница со списокм пользоватлей, где он може подтвердить или отклонить их
3. Пользователи могут удалять свои объявления, админ может удалять любые объявления 
4. Когда админ подтверждает регистрацию на почту юзеру приходит письмо


Требования к технической части:
1.	Для верстки использовать Bootstrap 4.
2.	База данных MySQL.
3.	Laravel
4.	JS - любые плагины, библиотеки и шаблонизаторы. Использовать возможности стандарта ES6
5.	Должны быть написаны миграции и сиды с несколькими тестовыми юзерами и объявлениями.
6.	При разработке использовать рекомендации из этой статьи https://github.com/alexeymezenin/laravel-best-practices
